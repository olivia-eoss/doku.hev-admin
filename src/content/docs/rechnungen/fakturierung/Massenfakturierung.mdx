In dieser Anleitung wird gezeigt, wie Sie eine Massenfakturierung starten können.

Sobald Sie sich im Dashboard befinden, sehen Sie bei "Abzurechnende Mitgliedschaften" alle ausstehenden Rechnungen, die Sie noch fakturieren müssen. Für eine detaillierte Übersicht klicken Sie auf "Details".
![massenfakturierung-uebersicht](/fakturierung/massenfakturierung-uebersicht.png)

1. Um die Massenfakturierung zu starten gehen Sie über das Menü (Häuschen, oben links) und suchen Sie in der Suche "Abrechnung starten".
2. Wählen Sie die betreffende Sektion aus, falls Sie auf mehrere Zugriff haben.
3. Bei der Produkt Kategorie können Sie die Leertaste im Feld drücken und Ihnen erscheinen zwei Möglichkeiten. Wählen Sie "Mitgliedschaften_Mitgliedschaften" aus.
   Beim Rechnungsdatum geben Sie das Datum an, welches auf der Rechnung erscheinen soll.
4. Sobald Sie diese Informationen ausgefüllt haben, können Sie die Massenfakturierung durch Klicken auf "Start" beginnen.
   ![massenfakturierung](/fakturierung/massenfakturierung.gif)

5. Sobald der Prozess fertig ist zeigt es Ihnen bei Offene Posten eine höhere Zahl an und bei PDF's zum Download erscheinen auch 1-3 mehr PDFs (1. normale Adressen, 2. Auslandsadressen und 3. Abweichender Rechnungspartner).
   ![massenfakturierung-anschauen](/fakturierung/massenfakturierung-anschauen.png)
6. Die Anzahl der "Abzurechnenden Mitgliedschaften" sollte nun auf Null stehen, es sei denn, es wurden während des Prozesses Neuaufnahmen gemacht. In diesem Fall werden sie hier angezeigt. Es gibt noch weitere Ausnahmen, bei denen dies vorkommen kann, die jedoch individuell betrachtet werden müssen.
   ![massenfakturierung-offene-posten](/fakturierung/massenfakturierung-offene-posten.png)
