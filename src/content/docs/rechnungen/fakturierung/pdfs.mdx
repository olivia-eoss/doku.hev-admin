Nach der Massenfakturierung werden dem User die im Rechnungslauf erstellten Rechnungen als PDFs zur Verfügung gestellt. Diese PDFs werden im Fenster "PDF" angezeigt. 
In der Standardeinstellung werden nur PDFs angezeigt, welche noch nicht als "Gedruckt" markiert wurden.

### PDFs als gedruckt markieren
PDFs werden vom System nicht automatisch als gedruckt markiert, dies kann vom User individuell gemacht werden. Um ein PDF als gedruckt zu markieren, gehen Sie wie folgt vor: 

1. öffnen Sie das Fenster PDF
2. wählen Sie das PDF aus, welches Sie als gedruckt markieren möchten
3. Markieren Sie die Checkbox "Gedruckt" durch anklicken
![PDF](/fakturierung/pdf.png)
4. Das PDF gilt im System als gedruckt

## Wichtig
Da der Druck des PDFs über Ihr normales PDF-Programm passiert, weiss der HEV Admin nicht, ob das PDF effektiv gedruckt wurde. Sie als User entscheiden, wann ein PDF gedruckt wurde. Die Markierung dient der Übersichtlichkeit und der besseren Organisation.
