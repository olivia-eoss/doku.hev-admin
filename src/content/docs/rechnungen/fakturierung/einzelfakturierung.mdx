Um Einzelrechnungen zu erstellen öffnen Sie das Fenster "HEV Rechnungsdisposition". Falls Sie die Einzelfakturierung im Rahmen einer Neuaufnahme durchführen, können Sie nach der Neuaufnahme direkt über die referenzierten Daten in die vorgefilterte Rechnungdisposition springen.

Um einzelne Rechnungen zu erstellen, können Sie die gewünschten Kandidaten über die verschiedenen Filter auswählen. Nach dem Sie alle Kandidaten gefunden haben, müssen Sie **zwingend im Fenster unten links durch zweimaliges klicken alle Kandidaten auswählen.** Danach können Sie über die Aktion "Auswahl fakturieren" die selektierten Kandidaten abrechnen. Zur Kontrolle wird Ihnen auf dem Button angezeigt, wie viel sie abrechnen.
