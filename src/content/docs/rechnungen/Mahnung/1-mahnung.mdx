# Mahnlauf

Um den Mahnlauf für Ihre Sektion zu starten, gehen Sie wie folgt vor:

## Fehlerhafte ESR-Zeilen abarbeiten

1. Prüfen Sie im Fenster 'ESR Zahlungsimport', ob alle Zahlungsfiles verarbeitet sind
2. Falls unklar ist, von welchem Mitglied eine Zahlung offen ist, kann diese noch offen stehen gelassen werden, damit diese zugeordnet werden kann, sobald sich das Mitglied meldet
3. Alle weitere Fälle sollten abgearbeitet werden, damit kein Mitglied unerwünscht gemahnt wird

Die Rechnungskandidaten werden immer erst über Nacht aktualisiert. Achten Sie also darauf, dass die Mahnungen erst einen Tag nach dem letzten ESR Import erstellt werden.

## Offene Posten vs. Mahndisposition

Bei Offene Posten erhalten Sie eine Übersicht von allen Rechnungen, welche noch aktuell offen sind. Achtung, es sind natürlich auch die aufgelistet, welche z.B. heute erstellt worden sind.
Wenn Sie das Fenster 'Mahndisposition' öffnen, erhalten Sie die Auflistung von allen Rechnungen, welche bereits über das Fälligkeitsdatum sind (30 Tage).

Falls Sie eine Mahnkarenz einfügen möchten für einzelne Mitglieder, damit diese nicht gemahnt werden beim Mahnlauf:

1. können Sie das entsprechende Mitglied im Filter suchen im Fenster 'Mahndisposition'
2. mit einem Doppelklick öffnen
3. im Feld 'Mahnkarenz' das gewünschte Datum eintragen

![Mahnprozess](/mahnung/mahnkarenz.png)

## Mahnlauf starten

Um den Mahnlauf zu starten, können Sie ins Menü, und hier im Suchfeld nach 'Mahnungen ' eingeben.
Sie erhalten ein Suchresultat mit 'Mahnungen verarbeiten und PDF-Druck auslösen'.

![Mahnprozess](/mahnung/Mahnlauf_starten.png)

Wählen Sie den Prozess aus, im nächsten Fenster können Sie den Prozess mit 'Start' beginnen.
Wichtig ist, wenn Sie auf mehrere Sektion Zugriff haben, dass Sie mit der entsprechender Sektion angemeldet sind, damit hier für die korrekte Sektion der Prozess gestartet wird.

![Mahnprozess starten](/mahnung/Mahnlauf_start.png)

Sobald hier der Prozess gestartet wurde, können Sie weiter im Programm arbeiten wie z.B. Mutationen vornehmen oder Neuaufnahmen erstellen, etc.
Wenn der Mahnprozess fertig ist, ist das PDF beim 'PDF' ersichtlich und Sie können diese Datei herunterladen. Falls im Mahnlauf ausländische Adressen vorkommen, wird hier ein extra PDF erstellt.
