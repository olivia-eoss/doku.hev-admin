Um eine einzelne Rechnugn zu drucken, gehen Sie wie folgt vor:

1. Öffnen Sie das Fenster "HEV Rechnungen".
2. Suchen Sie die Rechnung, die Sie drucken möchten.
3. Klicken Sie auf das Aktionsmenü (3 Punkte oben rechts).
4. Wählen Sie "Drucken" aus.
5. Es wird in einem neuen Browser-Tab die Rechnung angezeigt.

Bitte nachten Sie, dass auf diese Weise jeweils eine neue Version der Rechnung erstellt wird und nicht das Archiv-PDF der Rechnung ausgegeben wird.
