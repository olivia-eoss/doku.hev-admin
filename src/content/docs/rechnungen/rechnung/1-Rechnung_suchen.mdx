### Rechnungen suchen

Um bestehende Rechnungen zu suchen gehen Sie wie folgt vor:

1. Öffnen Sie das Fenster "HEV Rechnungen".
1. Nutzen Sie die beiden Filter Buttons um nach Rechnungen zu suchen. Der erste Filter grenzt nach Rechnungsdatum ein, im zweiten Filter können Sie mittels Standardfilter nach diversen Kriterien suchen.
1. Im Standardfilter können Sie nach Rechnungsnummer, Rechnung an (Mitglied) und nach Bezahlt J/N suchen.
1. Die Filter können beliebig kombiniert werden.

### Offene Posten FIltern

Um eine Liste mit allen offenen Rechnungen zu erhalten können Sie wie folgt filtern:

1. Öffnen Sie das Fenster HEV-Rechnung.
1. Öffnen Sie den Standardfilter.
1. Klicken Sie die Checkbox "bezahlt" 2x. Beim ersten Mal würden Sie nach "bezahlt = ja" suchen. Durch doppeltes klicken erhalten Sie einen grünen Rand um die Checkbox. Dies bedeutet "bezahlt = nein". Anschliessend auf "anwenden".
1. Klicken Sie unten links auf den Link "alle Zeilen dieser Seite auswählen" (durch einmaliges anklicken wählen Sie alle Namen auf der ersten Seite aus, durch Doppelklick wählen Sie alle gefilterten Mitglieder an).
1. Laden Sie die Daten über das Aktionsmenü (links oben, 3 Punkte) runter.
