In dieser Anleitung erklären wir Ihnen, wie Sie zur Statistik Einzelsektion gelangen können.

1. Klicken Sie auf das Symbol "Haus" (links oben), neben dem Dashboard und geben Sie in das Suchfeld "Statistik Einzelsektion (Jasper)" ein.

2. Es wird ein neues Fenster geöffnet. Geben Sie den Zeitraum an, in dem Sie die Statistik haben möchten.

3. Falls Sie Zugriff auf mehrere Sektionen haben, wählen Sie die entsprechende Sektion aus und klicken Sie auf "Start".

4. Anschliessend öffnet sich eine PDF- oder Excel-Datei.

![Bild_einzelsektion](/auswertung/einzelsektion.gif)

:::tip Hinweis

Im Falle, dass sich die Datei nicht öffnet kann es sein, dass Sie die Pop ups blockiert haben. In dem Fall klicken Sie in der Suchleiste auf das neu aufgetauchte Icon, um es zu akzeptieren.
![Bild_popup](/auswertung/popup.png) ![Bild_popup2](/auswertung/popup2.png)

:::
