## ESR File Upload

Im ersten Schritt müssen Sie das File, welches Sie von der Bank heruntergeladen haben, importieren. Dazu gehen Sie wie folgt vor:

1. Öffnen Sie das Fenster "ESR Zahlungsimport".
2. Erstellen Sie über das Aktionsmenü (links oben, drei Punkte) einen neuen Datensatz.
3. Ziehen Sie das File per Drag and Drop in den oberen Teil des Fensters. Sie sehen an dem grauen Hintergrund, dass das File korrekt platziert wurde. Durch loslassen der Maus laden Sie das File hoch.
   ![ESR File Upload](/esr/file_upload.png)
4. Danach importiert das System das File im Hintergrund und gleicht die Zahlungen mit den offenen Rechnungen ab. Sobald im ersten Register die Anzahl eingetragen wurde, ist der Upload beendet.
   ![ESR Upload beendet](/esr/esr_upload_beendet.png)
