# Neuaufnahme: Standardfall

In dieser Anleitung wird gezeigt, wie man neue Mitglieder in der eigenen Sektion aufnehmen kann. Diese Anleitung deckt den Standardfall ab.

### Einstieg

Im ersten Schritt öffnen Sie über das [Navigationsmenü](../tutorial-basics/navigation) das Fenster "Neue Mitgliedschaft".

![Gif Auswahl Menü Neue Mitgliedschaft](/neuaufnahme/neuaufnahme_1.gif)

### Dublettensuche

Für die Dublettensuche stehen Ihnen zwei Möglichkeiten offen. Sie können eine schnelle oder eine detaillierte Dublettensuche durchführen.

Um die schnelle Dublettensuche zu starten, können Sie einfach den Namen des neuen Mitglieds im Feld "Mitglied" eintippen. Wenn Sie ein Mitglied finden, können Sie das aus der Liste auswählen. Falls es keinen Treffer gibt, können Sie direkt eine neues Mitglied anlegen.
