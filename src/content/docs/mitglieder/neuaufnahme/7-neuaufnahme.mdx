# Neuaufnahme neues Mitglied

[In diesem Artikel wird die Dublettensuche und der Einstieg in die Neuaufnahme erklärt.](doublettensuche/)

Um die Neuaufnahme für ein neues Mitglied durchzuführen gehen Sie wie folgt vor:

1. Erstellen Sie einen neuen Kontakt.

![Gif Kontakt anlegen Neuaufnahme](/neuaufnahme/kontakt_anlegen_neuaufnahme.gif)

Zusätzlich können Sie eine Telefonnummer, einen Email-Adresse, den Haken für einen Newsletter und verschiedene Merkmale hinzufügen. Für die Erfassung von Doppelnamen nutzen Sie die [Erklärung zu dem Thema](../neuaufnahme/neuaufnahme_doppelname) .
Falls Sie mehrere Telefonnummern oder Email-Adressen erfassen möchten, können Sie diese nach der Neuaufnahme im Nutzer erfassen.


![Gif Adresse anlegen Neuaufnahme](/neuaufnahme/adresse_anlegen_neuaufnahme.gif)

2. Wählen Sie eine Mitgliederkategorie und eine Vertragsbedingung aus.

![Gif Abo anlegen Neuaufnahme](/neuaufnahme/abo_anlegen_neuaufnahme.gif)

Bei den Suchfeldern (mit dem "Lupe-Icon") können Sie mit einer Tastatureingabe den passenden Eintrag finden. Falls Sie den Namen des Eintrags nicht kennen, können Sie mit der Leertaste alle Einträge anzeigen lassen.
Mit Enter fügen Sie die Mitgliedschaft der Neuaufnahme hinzu. Alle zur Mitgliedschaft gehörigen Produkte werden automatisch hinzugefügt und mit der ausgewählten Vertragsbedingung versehen.

3. Schliessen Sie nun die Neuaufnahme ab, indem sie von Entwurf auf Fertigstellen wechseln.

![Gif Abo anlegen Neuaufnahme](/neuaufnahme/neuaufnahme_abschliessen.gif)

4. Springen Sie über die referenzierten Daten in die Rechnungsdisposition. In diesem Fenster werden die auf die Neuaufnahme gefilterten Rechnungskandidaten angezeigt.

![Gif Abo anlegen Neuaufnahme](/neuaufnahme/neuaufnahme_referenzierte_daten.gif)
