# Neuaufnahme mit Doppelnamen

Um ein Mitglied mit zwei Kontakt zu erfassen (Doppelnamen) gehen Sie grundsätzlich gleich vor wie bei einer normalen Mitgliedschaft.

### Start Neuaufnahme

Starten Sie die Neuaufnahme wie bei einer regulären Erfassung eines neuen Mitglieds. Details finden Sie unter [Neuaufnahme](/neuaufnahme/neuaufnahme)

Bei der Erfassung der Kontakte können Sie danach einen zusätzlichen Kontakt erfassen und beide als Doppelnamen kennzeichnen.

### Kontakte erfassen

Legen Sie beide Kontakte wie bei einer normalen Mitgliedschaft an. Die Reihenfolge spielt im Moment noch keine Rolle und kann nachträglich angepasst werden.

![doppelname](/neuaufnahme/doppelname.gif)

Achten Sie darauf, dass beide Kontakte den Haken bei "Doppelnamen" gesetzt haben müssen.

### Weitere Erfassung

Nach der Erfassung der Kontakte können Sie die Neuaufnahme wie hier beschrieben fertig machen. [Neuaufnahme eines neuen Mitglieds](/neuaufnahme/neuaufnahme)
