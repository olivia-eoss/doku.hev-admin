## Einstieg
Im ersten Schritt gehen Sie wie bei einer Standard Neuaufnahme vor und öffnen über das Navigationsmenü das Fenster "HEV Mitglieder". Von dort aus kann man über den grünen Button oben rechts eine Standard Neuaufnahme machen.

![Bild_zum-neuaufnahmefenster](/neuaufnahme/zum-neuaufnahmefenster.png)


## Erstellen Sie einen neuen Kontakt

## Adresse hinzufügen

![Bild_adresse-hinzufuegen](/neuaufnahme/adresse-hinzufuegen.gif)

Speichern Sie die Werbeadresse, indem Sie auf "Bestätigen" klicken.
