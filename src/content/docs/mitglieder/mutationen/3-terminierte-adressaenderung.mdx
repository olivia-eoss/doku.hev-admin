1. Öffnen Sie das Fenster HEV Mitglieder und suchen Sie das Mitglied.

2. Doppelklick auf die Zeile mit dem gesuchten Mitglied, um dieses zu öffnen.

![Bild_Sektionswechsel](/mutationen/adresse_mitglied_suchen.gif)

3. Wählen Sie anschliessend unter "Adresse" "Neu hinzufügen" an.

4. Fügen Sie die Adresse über das Feld "Edit" hinzu.

5. Wenn Sie die Adresse erfasst haben, klicken Sie aus dem Feld um diese zu speichern.

6. Wichtig ist, dass Sie beim nächsten Schritt bei "Gültig ab" das Datum eingeben, ab welchem die neue Adresse gelten soll.

![Bild_terminierte-adressaenderung](/mutationen/terminierte-adressaenderung.gif)
