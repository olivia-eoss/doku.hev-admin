In dieser Anleitung wird gezeigt wie Sie einen Brief erstellen können.

1. Öffnen Sie das Fenster "HEV Mitglieder".
2. Nutzen Sie hier die allgemeine Suche oder den Standard Filter um das entsprechende Mitglied zu suchen.
3. Öffnen Sie die Detailansicht vom Mitglied mit einem Doppelklick.

![Bild_Sektionswechsel](/mutationen/adresse_mitglied_suchen.gif)

Sobald Sie im Mitglied sind gehen Sie auf die drei Punkte und bei HEV Mitglieder auf "Letter".
EsSobald Sie im Mitglied sind, gehen Sie auf die drei Punkte (oben links) und bei HEV Mitglieder (Spalte links) auf "Letter". Es wird sich ein neues Fenster öffnen und dort können Sie über den weissen Balken die Art von Brief suchen, indem Sie ihn Anklicken.

![Bild_brief-machen](/mutationen/brief-machen.gif)

Sie können nun über das Druckersymbol den Brief als PDF öffnen und ihn z.B. ausdrucken usw. Über den grünen Button "create" kommen Sie zurück in das Mitgliederfenster und der Brief wird dort in der Kommunikation gespeichert.

![Bild_brief-speichern](/mutationen/brief-speichern.gif)

:::tip Hinweis

Fals Sie ein Sektionsspezifischen Brief benötigen, können Sie gerne auf uns zu kommen und wir schalten dies für Sie auf.

:::
