Eine bestehende Mitgliedschaft können Sie wie folgt kündigen:

1. Öffnen Sie das Fenster HEV Mitglieder und suchen Sie das Mitglied.

2. Doppelklick auf die Zeile mit dem gesuchten Mitglied, um dieses zu öffnen.

3. Öffnen Sie das Aktionsmenü (links oben, drei Punkte) und wählen Sie in der rechten Spalte des Menüs die Option “Vertrag kündigen”.

4. Tragen Sie das Kündigungsdatum, den Kündigungsgrund und, wenn vorhanden eine Nachricht ein.

5. Wählen Sie die Checkbox “Offene Rechnungen gutschreiben” an, falls offene Rechnungen bestehen und Sie auf eine Begleichung des offenen Betrages verzichten. Falls das Mitglied die Rechnung trotz Kündigung begleichen soll, lassen Sie die Checkbox leer. Wählen Sie die Checkbox "Rechnungskandidat schliessen" an, wenn noch keine Rechnung erstellt worden ist. Somit verhindern Sie das eine Rechnung erstellt wird. 

6. Starten Sie den Prozess rechts oben mit dem Button “Start”.
