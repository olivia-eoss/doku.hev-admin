1. Öffnen Sie das Fenster "HEV Mitglieder".

2. Nutzen Sie hier die allgemeine Suche oder den Standard Filter um das entsprechende Mitglied zu suchen.

3. Öffnen Sie die Detailansicht vom Mitglied mit einem Doppelklick.

![Bild_Sektionswechsel](/mutationen/adresse_mitglied_suchen.gif)

4. Wählen Sie über das Aktionsmenü (ganz links, 3 Punkte) die Aktion “Abo-Rechnungspartner ändern” aus.

5. Wenn Sie die Aktion auswählen, öffnet sich ein Prozess, in welchem drei Werte eingetragen werden müssen:

Im ersten Feld tragen Sie den Rechnungsempfängers ein. Je nachdem auf wie viele Mitglieder Sie Zugriff haben, dauert die Suche etwas länger. Um zu suchen tippen Sie am einfachsten einfach die Mitglieder-Nummer ein. Wenn Sie den Partner ausgewählt haben, kann der Rechnungsstandort ausgewählt werden. Dazu klicken Sie in das jeweilige Feld und drücken die Leertaste. Dann werden alle verfügbaren Adressen angezeigt und Sie können die passenden Daten auswählen. Es kann auch vorkommen, dass nur ein "." erscheint. Dann bitte diesen anwählen. 
Wichtig ist dabei, dass der Rechnungspartner schon im Voraus erfasst wurde.

6. Führen Sie den Prozess mit dem Button “Start” aus.

  ![Bild_rechnungspartner](/mutationen/rechnungspartner.gif)
