Falls ein Mitglied eine Zeitung nicht erhalten möchte, können Sie den Versand unterdrücken, in dem Sie die Anzahl auf 0 setzen. Kündigen Sie die Zeile der Zeitung nicht. Um die Anzahl auf 0 zu setzen, können Sie wie folgt vorgehen:

1. Öffnen Sie das Fenster HEV Mitglieder und suchen Sie das Mitglied.

![Bild_neuaufnahme-bestehendes-mitglied](/mutationen/neuaufnahme-bestehendes-mitglied.png)

2. Doppelklick auf die Zeile mit dem gesuchten Mitglied, um dieses zu öffnen.

2. Wählen Sie aus dem Register "Abos" die Zeile der Zeitung aus, für welche Sie die Menge ändern möchten.

![Menge ändern](/mutationen/menge_aendern_neu.png)

5. Starten Sie den Prozess "Menge ändern" über das Aktionsmenü (ganz links die 3 Punkte (senkrecht); neben dem Häuschen).

![Menge ändern Schritt 2](/mutationen/menge_anedern_neu_2.png)

6. Tragen Sie die Anzahl "0" ein und bestätigen Sie mittels Klick auf "Start".

Zurück im Mitglied sehen Sie, dass die Anzahl Null ist.

![Bild_im-mitglied](/mutationen/im-mitglied.png)
