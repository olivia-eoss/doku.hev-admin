Wenn ein Mitglied mehrere Adressen hat, ist es teilweise schwierig zu sehen, welche Adresse für welche Funktion verwendet wird. Daher hat jede Adresse zwei Checkboxen:

![Adressen Funktionen](/mutationen/adressen_standards.png)

### Lieferadresse
Wenn eine Adresse als Lieferadresse gekennzeichnet ist, wird diese für den Versand der Zeitung verwendet. 

### Rechnungsadresse
Wenn eine Adresse als Rechnungsadresse gekennzeichnet ist, wird diese für den Versand der Rechnung verwendet. 

### Änderungen
Sie können durch ändern der Checkboxen auch einfach die Adresse ändern. Wenn Sie also eine neue Adresse z.B. als Lieferadresse markieren, wird diese automatisch für den Zeitungsversand verwendet. 
Wenn eine Adresse keinen Haken gesetzt hat, wird sie im System nicht verwendet.
